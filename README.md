# Previsão de Churn em Serviços de Streaming
Projeto desenvolvido para a disciplina de Inteligência Artificial da UniFECAF.

##  Objetivo
Prever quais clientes têm maior probabilidade de cancelar o serviço (churn), utilizando modelos de Machine Learning no Orange Data Mining.

##  Arquivos do Projeto
- **Relatório Analítico (PDF)** – Explicação teórica e prática do projeto  
- **base_clientes.csv** – Base de dados fictícia com 100 clientes  
- **projeto_orange.ows** – Arquivo do Orange Data Mining com todo o fluxo  
- **Vídeo Pitch** – Apresentação de até 4 minutos explicando o projeto  

##  Modelos Utilizados
- Logistic Regression  
- Random Forest  

Ambos avaliados com:
- Test & Score  
- Matriz de Confusão  
- Métricas Individuais  
- Curva ROC  

##  Resultados
Os dois modelos atingiram desempenho perfeito:
- Acurácia: 1.000  
- Precisão: 1.000  
- Recall: 1.000  
- F1-score: 1.000  
- AUC: 1.000
